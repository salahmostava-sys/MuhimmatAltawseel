## صفحة 2: لوحة التحكم (الرئيسية)

**التصميم**: صفحة مع 4 تبويبات أعلى

### تبويب النظرة العامة:
- **صف 6 بطاقات KPI** أعلى: إجمالي الطلبات، متوسط/مندوب، تحقيق الهدف (% مع progress)، أفضل مندوب اليوم، متوسط التقييم، التنبيهات
- **قسمين جنب بعض**: يسار = لوحة تحليلات ذكية (AI insights مع توصيات)، يمين = مقارنة شهرية + أسبوعية (بطاقات بأرقام ونسب تغيّر)
- **بطاقات المنصات**: بطاقة لكل منصة توصيل (اسم بلون المنصة + طلبات + مناديب + هدف + نمو%)
- **حسب المدينة**: مكة / جدة مع أرقام
- **تنبيهات ذكية**: بطاقات تحذيرية (مندوب متراجع، مندوب غائب)
- **جدول تفصيلي** أسفل: كل المناديب مع أدائهم

### تبويب التحليلات:
- **رسم بياني أعمدة**: اتجاه الطلبات آخر 6 أشهر
- **رسم بياني خطي**: الطلبات يوم بيوم
- **مقارنة المنصات**: كل منصة بنسبة نموها
- **توزيع الأداء**: ممتاز/متوسط/ضعيف مع progress bars

### تبويب التصنيف:
- **4 أقسام**: أفضل 10، أسوأ 10، أكبر تحسّن، أكبر انخفاض
- كل مندوب: اسم + طلبات + نمو% + اتجاه

### تبويب المنصات:
- **بطاقة لكل منصة**: طلبات، مناديب، progress bar تحقيق الهدف
- **رسم بياني أعمدة مزدوج**: الشهر الحالي vs السابق لكل منصة
- **جدول مقارنة** قابل للترتيب (8 أعمدة)

## صفحة 3: الموظفون

- **شريط أدوات**: بحث + فلاتر (مدينة/حالة/وظيفة) + أزرار (إضافة، تصدير Excel، استيراد، طباعة) + تبديل أعمدة مرئية
- **جدول كبير**: اسم + صورة مصغرة، هاتف، مدينة، حالة الكفالة (badges ملونة)، حالة الرخصة، تاريخ الإقامة (ألوان حسب القرب من الانتهاء)، المنصة المربوطة، إجراءات (عرض/تعديل/حذف)
- **الخلايا قابلة للتعديل inline**: اضغط على خلية → تتحول لحقل إدخال
- **pagination** أسفل الجدول
- **modal إضافة/تعديل**: متعدد الخطوات (بيانات أساسية → مستندات → صورة)
- **صفحة تفصيلية**: عند الضغط على موظف → صفحة ملف شخصي كامل

## صفحة 4: الحضور والانصراف

**تبويبان:**

### التسجيل اليومي:
- **تقويم** لاختيار التاريخ (يسار)
- **قائمة موظفين** مع dropdown لكل واحد (حاضر ✅ / غائب ❌ / إجازة / مريض / متأخر)
- **ألوان**: أخضر=حاضر، أحمر=غائب، أصفر=إجازة، أزرق=مريض، برتقالي=متأخر

### السجل الشهري:
- **جدول شبكة**: الموظفين (صفوف) × أيام الشهر (أعمدة)
- ملخص لكل موظف

---

## صفحة 5: الطلبات

**3 تبويبات:**

### الطلبات (Spreadsheet):
- **شبكة ضخمة**: المناديب (صفوف) × أيام الشهر (أعمدة)
- **tabs منصات** أعلى: الكل / أمازون / هنقرستيشن / إلخ (كل tab بلون المنصة)
- كل خلية = عدد الطلبات (رقم قابل للتعديل)
- إجماليات لكل صف وعمود
- شريط أدوات: بحث + استيراد Excel + تصدير + طباعة + حفظ + قفل الشهر

### الدوام:
- نفس الشبكة لكن الخلايا = dropdown (حاضر ✅ / غائب ❌ / فاضي)
- الألوان: أخضر=حاضر، أحمر=غائب، رمادي=لم يُحدد
- المجموع: أيام الحضور / إجمالي الأيام

### ملخص الشهر:
- بطاقات: إجمالي طلبات + مناديب + هدف لكل منصة
- جدول ملخص: كل مندوب × كل منصة = طلبات

---

## صفحة 6: الرواتب

- **بطاقات ملخص** أعلى: صافي الرواتب الإجمالي + طلبات/دوام لكل منصة
- **جدول رواتب ضخم** (أهم صفحة):
  - كل صف = موظف
  - أعمدة ثابتة: الاسم، المدينة، أيام العمل، حالة الاعتماد
  - **أعمدة ديناميكية لكل منصة** (بلون المنصة): طلبات/دوام + راتب المنصة
  - أعمدة: إجمالي الراتب، الحوافز، الخصومات، السلف، صافي الراتب
  - **الخلايا قابلة للتعديل**: حوافز، خصومات يدوية
  - **أزرار**: اعتماد، كشف راتب PDF
- **شريط أدوات**: بحث + فلتر حالة + عرض (جدول/بطاقات) + تصدير Excel + تصدير PDF جماعي + طباعة + استيراد

## صفحة 7: السلف

- **جدول**: الموظف، المبلغ، التاريخ، عدد الأقساط، الحالة (نشط/مكتمل/متوقف)، المتبقي
- **modal إضافة سلفة**: اختيار موظف + مبلغ + أقساط + تاريخ
- كل سلفة قابلة للتوسيع → جدول أقساط (تاريخ + مبلغ + حالة)
- فلاتر: بحث + حالة

## صفحة 8: المصاريف والإيرادات

**تصميم عمودين:**
- **3 بطاقات أعلى**: إجمالي الإيرادات (أخضر) + إجمالي المصاريف (أحمر) + الفرق/الرصيد (أخضر=ربح / أحمر=خسارة)
- **العمود الأيسر (💰 الإيرادات)**: جدول + صف إضافة inline (dropdown تصنيف + حقل مبلغ + تاريخ + زر إضافة أخضر)
- **العمود الأيمن (💸 المصاريف)**: نفس الهيكل بس أحمر + الرواتب تظهر تلقائياً بعلامة 🔒
- تصنيفات الإيرادات: توصيل، منصات، عقود، أخرى
- تصنيفات المصاريف: رواتب، إيجار، وقود، صيانة، تأمين، اتصالات، مستلزمات، إدارية، أخرى

## صفحة 9: استهلاك المناديب (الوقود)

**3 عروض (tabs):**

### عرض شهري:
- 4 بطاقات: كيلومترات + بنزين + تكلفة/كم + طلبات
- جدول: المندوب + أيام + كم + بنزين + تكلفة/كم + الدباب + طلبات + بنزين/طلب

### عرض يومي:
- قائمة مناديب accordion (اضغط → يفتح سجلات يومية)
- صف إضافة inline أسفل

### جدول بيانات (Spreadsheet):
- **شبكة**: المناديب (صفوف) × أيام الشهر (أعمدة)
- **3 قيم في كل خلية**: 📦 طلبات (أزرق) + 🛣️ مسافة (أخضر) + ⛽ بنزين (أصفر)
- هيدر مزدوج: يوم الأسبوع + رقم اليوم (الجمعة بلون وردي)
- إجماليات صف + عمود
- 6 بطاقات إحصائية أعلى
